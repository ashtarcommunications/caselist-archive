There’s quite a bit of Aff intell to sort through for MSU.
Here – I think – is the most useful stuff I could say in the overview:

First – Every MSU team has run some version of the biofuel Aff all year.

Secondly – We’ve honestly run more than 20 different plan texts this semester.

Instead of listing all of them– let me start by rattling off the relevant distinctions:
n Sometimes the plan lifts the tariff, sometimes it does not. Usually it does.
n The plan has always lifted a large percentage of subsidies for corn ethanol
n Sometimes the plan has also lifted subsidies for biodiesel. 50-50%.
n Sometimes the plan phases-out subsidies, sometimes it does not. Usually it has.
n Sometimes how this phase-out will take-place has been delineated to the point of insanity (dealing with opponents into vagueness args). Sometimes it does not delineate to this extreme. Usually it does not.
n Sometimes the word “corn” has been used – sometimes is has not. 50-50%.
n Sometimes the word “biodiesel” has been used – we have also used the phrases:
o Agri-fuel
o Agri-diesel
o Agro-fuel
o Agro-diesel
o non-petroleum-based fuels consisting of short chain alkyl (methyl or ethyl) esters, made by transesterification of tallow, which can be used in unmodified diesel-engine vehicles
n Sometimes the subsidies that are lifted have been delineated to the point of insanity (same vagueness crowd). Usually it has not been. When it has done so it has (depending on the situation) eliminated the following subsidies:
o Volumetric Ethanol Excise Tax Credit
o Volumetric Biodiesel Excise Tax Credit
o Small Ethanol Producer Credit
o Production credits
n Very few drafts of the plan have ever used the phrase “nearly all” – though some have b/c we did that at some of the debates @ GSU.
n We have read a version of the plan that dodges the I/thou language args.
n We have specified the Congress as an actor.
n We have used verbs other than “phase-out (to the point of elimination)” and “eliminate”. Most notably, we “rescinded”. We did this in exactly one debate.
n We have done totally trivial things like have a period at the end of the plan and not have a period at the end of the plan.

Here is an example of the plan text that we most-often read in the first semester:
The United States Federal Government should phase-out (to the point of elimination) its:
n domestic subsidies for the blending and production (including associated plant development) of biodiesel and corn ethanol
n Ethanol Import Tariff
